# FitTracker Mobile (iOS)

Questa è una versione web mobile di FitTracker pensata per iOS. Puoi aprire `index.html` nel browser del tuo iPhone o iPad. Per un utilizzo migliore, salva la pagina nella schermata Home tramite Safari.

## Come usare

1. Apri `index.html` in Safari su iOS
2. Registrati con email e password
3. Accedi
4. Avvia il timer e completa l'allenamento

## Compatibilità
- Browser moderni su iOS
- Puoi aggiungere l'app alla schermata Home come PWA leggera
